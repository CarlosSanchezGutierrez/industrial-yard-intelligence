# Namiki UX polish P0.5

## Purpose

This pass connects material detail with operational history and evidence.

## Added

- Material detail panel now shows related history.
- Material detail panel now shows linked evidence.
- Selected material row is highlighted.
- Capture page now includes an evidence workspace.
- Evidence workspace supports:
  - material selection
  - evidence type
  - note
  - source suggestion
  - demo evidence queue

## Evidence sources represented

- GPS del dispositivo
- Celular / supervisor
- Drones RTK (Real Time Kinematic)
- Bastones topográficos con trípode

## Why

The system should feel like an operational product, not just a dashboard.

A material must connect naturally to:

- location
- status
- events
- evidence
- review actions
- field capture
