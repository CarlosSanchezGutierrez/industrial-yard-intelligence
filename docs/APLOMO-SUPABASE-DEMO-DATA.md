## Orden correcto

1. Ejecutar `docs/sql/aplomo_schema_v1.sql`.
2. Ejecutar `docs/sql/aplomo_seed_demo_v1.sql`.
3. Revisar tablas en Supabase.
4. Confirmar que existe una captura GPS demo.
5. Después conectar frontend.
