# Namiki UX polish P0.4

## Purpose

This pass adds deeper interaction to the professional console.

## Added

- Map layer controls:
  - Zonas
  - Rutas
  - Materiales
  - Equipo
- Map operational summary cards.
- Timeline search.
- Timeline priority filter.
- Timeline event type filters.
- Timeline result summary.
- Empty state when filters return no events.

## Why

The console must feel controllable and operational.

The map should not be a static image.
The timeline should not be a static list.
