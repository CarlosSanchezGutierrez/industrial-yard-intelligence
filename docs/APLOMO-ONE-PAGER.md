# Aplomo Systems — One-pager

## Qué es

Software para patios industriales.

Consola visual para controlar materiales, GPS, evidencia y operaciones en campo.

## Para quién

- Puertos.
- Cementeras.
- Minería.
- Construcción.
- Acereras.
- Canteras.
- Terminales logísticas.
- Patios ferroviarios.
- Inmobiliarias industriales.
- Empresas con operación física intensiva.

## Problema

Las operaciones de patio suelen depender de registros manuales, mensajes, fotos aisladas, memoria del personal y comunicación informal.

Eso causa incertidumbre sobre ubicación de materiales, poca trazabilidad, errores operativos y pérdida de tiempo.

## Solución

Aplomo permite convertir el patio físico en una consola digital:

- Mapa.
- GPS.
- Puntos.
- Perímetros.
- Materiales.
- Evidencia.
- Responsables.
- Historial.
- Estado de capturas.
- Sincronización futura campo-supervisión.
