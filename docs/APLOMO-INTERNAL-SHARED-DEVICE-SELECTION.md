# Selección compartida de dispositivo — Aplomo Systems

## Objetivo

Permitir que el mapa, tablas, detalle técnico y perfil de integración industrial trabajen sobre el mismo dispositivo seleccionado.

## Qué cambia

Ahora se puede seleccionar un dispositivo desde:

- Punto del mapa.
- Tabla de posiciones vivas.
- Tabla de dispositivos.
- Tabla de eventos gobernados.
- Selector del panel administrativo.
- Selector del detalle de dispositivo.

## Resultado

Al seleccionar un dispositivo:

- El punto del mapa se resalta.
- El detalle técnico cambia.
- El perfil industrial cambia.
- Las tablas quedan alineadas al runtime compartido.

## Por qué importa

Esto acerca el panel interno a una consola SaaS real para administradores.

Un administrador puede pasar de vista general a diagnóstico específico sin perder contexto.
